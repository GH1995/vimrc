```

```

# How to use

My leader is `-`, and you can modify it in ~/.vim/vimrc/lab.vim.

| key          | use      |
|:------------:|:---------|
| `<leader>ut` | UndoTree |

## Markdown

| key                           | use                                               |
|:-----------------------------:|:--------------------------------------------------|
| `<leader>tf`                  | format table                                      |
| `:toc`                        | open the toc                                      |
| `<leader>hi` and `<leader>hd` | increase and decrease header(cursor on the table) |
